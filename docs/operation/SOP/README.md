## Summary
--------------------------------------------------------------------------------

This is the index for SOPs for various tasks handling the Arriba WGTS RNA Pipeline.

These tasks range from:
* Installation / deployment of a new CWL Workflow onto the ICAv2 platform
* Verifying a new version of the Arriba WGTS RNA Pipeline
* Troubleshooting common issues with the Arriba WGTS RNA Pipeline
* Kicking off a reanalysis of a set of samples manually
